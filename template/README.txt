- Instalar node e npm
	nodejs.org

- Instalar dependencias globais via npm
	npm install grunt
	npm install grunt-cli
	npm install sass
	
- Instalar dependencias locais via npm
	npm install

- Gerar modernizr na pasta js
	modernizr -c modernizr-config.json -d js/ -u

- Copiar arquivos da node_modules
	grunt build

- Iniciar servidor
	npm run server

- Iniciar o watch
	grunt watch

- Compilar o scss
	grunt sass
